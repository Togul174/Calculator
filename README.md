Калькулятор для выполнения простых арифметических задач.

Функционал:
1. Сложение, вычетание, умножение, деление.
2. Поддержка десятичных дробей и смены знака.
3. Ввод с клавиатуры, либо нажатием на предложенные клавиши.
4. Очистка поля и последней набранной цифры.
5. Защита от деления на ноль.

Технологии:
1. HTML - разметка.
2. CSS - стилизация и адаптивность.
3. JavaScript - логика обработки вычислений и ввода.
4. Webpack - сборка проекта.

Установка и запуск:
1. npm i - установка зависимостей.
2. npm run build - сборка проекта. 
3. npm start - запуск проекта.
